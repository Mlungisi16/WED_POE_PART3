FreshKickers Website

Project Overview:
FreshKickers is a website for a sneaker cleaning and customization business. It provides information about the company, its services, news updates, gallery of sneaker designs, contact details, location, and an enquiry form for customers. The website features a modern, responsive design with interactive elements such as service selection, price estimation, login/signup modals, and a WhatsApp chat button.

Features

Responsive Navbar

Present on all pages for easy navigation.

Links to Home, About, Services, Contact, News, Location, Gallery, Enquiry pages.

Home Page (home.html)

Welcome message and homepage image.

Cards highlighting main services: Sneaker Wash and Custom Designs.

Login and Signup modals with form validation.

Cookie consent bar.

About Page (about.html)

Company description and mission.

Images of team and work examples.

Tagline: “Walk it like you talk it.”

Services Page (services.html)

Sneaker Wash and Custom Design options.

Dropdowns for service type, wash type, pickup/delivery option.

Price calculation based on service selection.

Login required to book services.

File upload for custom designs.

Gallery Page (gallery.html)

Displays images of sneaker transformations and custom designs.

Multiple images presented in a grid layout.

News Page (news.html)

Shows latest news and updates for FreshKickers.

Includes images related to announcements.

Contact Page (contact.html)

Shows phone number, email, and physical address.

Embedded Google Map for store location.

Location Page (location.html)

Detailed store location with a Google Map embed.

Information about service availability across South Africa.

Enquiry Page (enquiry.html)

Form for submitting enquiries with validation for name, email, phone, and message.

WhatsApp chat button for direct communication.

Technologies Used

HTML5

CSS3 (External stylesheet CSS/style.css + inline styling for specific pages)

JavaScript for interactivity (modals, form validation, price calculation)

Google Maps embed

WhatsApp chat integration
